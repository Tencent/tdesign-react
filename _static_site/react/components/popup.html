基础弹出层 
由是让浮层内容和触发元素组成，两者均可自定义。使用  自定义浮层内容。
触发元素 
可以使用  自定义触发元素。
不同触发方式的弹出层 
提供悬浮时触发（默认）、点击时触发、获取焦点时触发、右击时触发等方式。
位置方向 
使用  控制浮层方向，如果需要浮层箭头，设置  即可。
浮层样式 
浮层样式可以使用 overlayClassName 或者 overlayStyle 定义。
1、  用于定义浮层样式类名。
2、  用于定义浮层样式，比如浮层宽度。浮层宽度默认根据内容宽度呈现，可自由设置宽度和最大宽度。值为类型为函数时，可以实现浮层宽度和触发元素同宽。
可控制显示的弹出层 
可以通过  自由控制弹出层的显示或隐藏。
挂载父节点 
浮层默认父节点是 ，可通过  自由调整挂载的父节点元素。
可隐藏时销毁的弹出层 
 用于控制浮层隐藏时是否销毁浮层内容。
禁用状态的弹出层 
组件禁用后，不再显示弹出层。
