      何时使用 
      

      需要提醒用户来自系统的消息，且不打断用户；
      

      带有解释描述的提醒内容。
      

      与页面布局相关 
      

      消息通知建议悬浮出现在页面右上角、左下角或右下角，避免对页面主要内容的干扰。

消息通知
需要提醒用户来自系统的消息，且不打断用户；带有解释描述的提醒内容。

      

      
全局提示
通常跟随在用户操作之后，对用户的操作作出轻量、即时的反馈；自动消失，不带其他交互操作；反馈内容简短。
