可过滤的选择器 
输入过滤已选项。在有特定诉求的业务场景时使用。默认过滤文本，有  方法时，按照  方法过滤。
1、和()存在时，执行远程搜索。
2、存在时（不需要有），执行方法。
3、仅有时，执行默认过滤 不区分大小写。

      何时使用 
      

      需要在有限的空间展示大量选项，并从中选取单个或多个选项时。
